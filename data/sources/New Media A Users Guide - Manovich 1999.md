# New Media: A User's Guide

## Summary

A systematic overview of new media's origins and principles, later absorbed into *The Language of New Media*. Manovich traces the parallel histories of modern media (Daguerre, Lumière) and computing (Babbage, Hollerith, Turing, Zuse) and identifies four key principles of new media: discrete representation, numerical representation, automation, and variability.

## Key Arguments

- **Parallel histories converge**: Media technologies (daguerreotype, cinema) and computing (Analytical Engine, tabulating machines) developed in parallel from the 1830s, serving complementary functions of mass society — mass media for ideological dissemination, computers for data processing. They converge when Zuse punches binary code onto 35mm film.
- **Turing Machine as film projector**: Turing's 1936 design — reading/writing on endless tape — "looks suspiciously like a film projector." Both cinema and computing needed suitable storage media and data coding methods.
- **Jacquard's return**: The computer returns to its origins as Jacquard's loom — "a media synthesizer and manipulator," not just an Analytical Engine for numbers.
- **Four principles**: (1) Discrete/modular representation at every scale ("fractal structure"); (2) Numerical representation enabling mathematical description and algorithmic manipulation; (3) Automation of creation, manipulation, and access (low-level: filters, templates; high-level: AI-driven); (4) Variability — new media objects exist in potentially infinite versions.
- **Variability as key principle**: Connects new media logic to post-industrial "production on demand." Old media = industrial mass society (identical copies); new media = post-industrial society (customized versions). "Today descendants of the Jacquard's loom, the Hollerith tabulator, and Zuse's cinema-computer are now working to convince us that we are all different."
