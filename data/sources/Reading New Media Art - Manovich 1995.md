# Reading New Media Art

## Summary

A short exhibition review (NEWFOUNDLAND II) that addresses a fundamental problem of media art: the tension between aesthetic contemplation and the instrumental behaviors (problem solving, learning interfaces, experimentation) that interactive works demand. Manovich surveys several works that offer different solutions.

## Key Arguments

- **The core problem**: Media art requires the viewer to simultaneously contemplate aesthetically and learn to "use" a work — reading instructions, navigating interfaces, solving interaction puzzles. Can these be combined?
- **Solutions observed**:
  - Avoiding interactivity altogether (Waliczky's *The Way* — pure 3D animation with custom perspectival systems)
  - Using a familiar interface (Iwai's *Piano — As an Image Media* — the piano keyboard as interface)
  - Integrating multiple media languages (Boissier's *Flora Petrinsularis* — the loop as poetic element; Courchesne's *Portrait One* — combining graphic design and cinematic identification)
- **The loop resurrected**: Boissier's piece uses the loop — which gave birth to pre-cinema and was then banished to cartoons — as a fundamental element of multimedia language capable of carrying rich meaning.
- **Collective vs. individual viewing**: Some installations create a new hybrid: one viewer interacts (via joystick, mouse, head sensor) while others watch, creating "a combination of individualized and collective spectatorship."
- **EVE installation**: A panoramic VR space that shows the outside reality rather than a simulated world. The person wearing the sensor becomes simultaneously master subject, visionary, and tool/interface for others.
