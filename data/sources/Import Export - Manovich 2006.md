# Import/Export: Design Workflow and Contemporary Aesthetics

## Summary

An essay arguing that the import/export commands enabling compatibility between software applications are historically more important than the individual operations these programs offer. The cross-application workflow — creating elements in one program, importing into another, combining with elements from yet another — is the material basis of contemporary hybrid visual aesthetics.

## Key Arguments

- **Import/export as key operation**: Just as import/export business is >60% of greater LA's economy, compatibility between file formats is the essential condition enabling contemporary design logic.
- **Pre-software separation**: Before software, filmmakers, graphic designers, and animators used completely different technologies and inevitably created differently looking images. Cinema showed photorealistic images; graphic design featured typography and abstract elements; cartoons showed hand-drawn characters.
- **Software-based workflow**: Designers use a small set of tools (Photoshop, Illustrator, After Effects, Maya, etc.) and move elements between them. The final program depends on output media, but the process always involves multiple applications.
- **"Design Across Media"**: Adobe Creative Suite's phrase accurately describes both capabilities and actual use. Elements can be reused across different project types (magazine ad → video → web site).
- **Two consequences**: (1) Visual aesthetics of hybridity — typical design combines techniques from multiple media; (2) Same techniques and strategies appear across all output media regardless of project type.
- **Deep remixability vs. multimedia**: If software only combined elements, we'd see "multimedia." Instead, the "deep" interactions between working methods produce genuinely new hybrid techniques. "A lens blur filter" applied to non-photographic content is an example.
- **Parametric design**: Software enables compositions from hundreds of variations of the same element — replacing modernism's completely regular grids with more complex structures (exemplified by Zaha Hadid's variable grids).
