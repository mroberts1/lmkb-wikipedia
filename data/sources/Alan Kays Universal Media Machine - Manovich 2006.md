# Alan Kay's Universal Media Machine

## Summary

A foundational essay for Manovich's software studies project, tracing how Alan Kay and collaborators at Xerox PARC in the 1970s turned the computer into a "metamedium" — not just simulating existing media but systematically augmenting them with new properties. Manovich argues against the "remediation" framework (Bolter/Grusin), showing that computational media was conceived from the start as fundamentally new, not merely a representation of old media.

## Key Arguments

- **Against remediation**: While digital media may *look* the same as its predecessors on the surface, it *functions* in fundamentally different ways. A digital photograph can be modified, shared, inserted into other documents, algorithmically enhanced — properties that make it "radically different from its traditional equivalent."
- **Simulation + augmentation**: Kay's group didn't just simulate existing media — they added "many new properties." A simulated book gains dynamic search, non-sequential access, multiple views. This is "magical paper," not just digital paper.
- **Engelbart's "view control"**: One of the most fundamental yet undiscussed new techniques — the ability to switch between many different views of the same information. Demonstrated in his 1968 demo and now embedded in all media software.
- **Permanent extendibility**: "New media is 'new' because new properties (i.e., new software techniques) can always be easily added to it." Software separates hardware from functionality, making experimentation a default feature. The computer is structurally "avant-garde."
- **Computer as metamedium**: Kay and Goldberg's key formulation: the computer is "a metamedium whose content is a wide range of already-existing and not-yet-invented media." It is simultaneously a set of different media AND a system for generating new media tools.
- **No single language**: Unlike cinema or print, which gradually found their own languages, computational media keeps expanding its capabilities. "We are gradually discovering that the computer can speak more and more languages."
- **Smalltalk and user programming**: Kay's vision included users being able to write their own media tools — "literacy" means both reading (accessing) and writing (generating) in a medium. Processing is a contemporary realization of this vision.
