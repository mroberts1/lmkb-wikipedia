# Software Studies Index

Topic index for Manovich's software studies work — analyzing how specific software tools shape cultural production, aesthetics, and experience. This strand emerges in the late 1990s and becomes dominant in his 2004-2007 writing.

## Core Concepts

- [[Software Studies]] — The field: software as cultural engine
- [[Metamedium]] — The computer as medium of media (Alan Kay)
- [[Deep Remixability]] — Remixing techniques, not just content
- [[Hybrid Media]] — New media species from technique combination
- [[Compositing]] — Layering as the key operation of digital visual culture
- [[Velvet Revolution]] — The quiet transformation of moving image culture (1993-98)
- [[Post-media Aesthetics]] — Aesthetics after medium specificity
