# Metamedium

## Definition

The concept, originating with Alan Kay and Adele Goldberg (1977), that the computer is not just another medium but a "metamedium" — a medium that can simulate all existing media while also enabling the creation of entirely new media that have no physical precedents. Manovich adopts and extends this concept as the theoretical foundation for [[Software Studies]].

## Core Thesis

The computer does not merely "remediate" old media (as Bolter and Grusin argue). It *simulates* them — reproducing their properties algorithmically — and then *augments* them with new capabilities impossible in the original physical medium. A simulated photograph can be painted on; a simulated film can have infinite layers; simulated typography can animate in 3D. This simulation-plus-augmentation produces fundamentally new media species, not just digital versions of old ones.

## Key Principles

- **Simulation, not remediation**: Manovich explicitly rejects Bolter and Grusin's "remediation" framework. The computer doesn't just reference old media — it computationally models their properties, then extends them
- **Augmentation**: Each simulated medium gains new properties: undo, layers, filters, algorithmic generation, programmability. These additions are not cosmetic but change the medium's fundamental nature
- **Permanent extendibility**: Unlike any physical medium, the metamedium can be infinitely extended through new software. Users and developers continuously add new tools, plug-ins, and operations — the medium never reaches a fixed state
- **New media species**: The metamedium enables media types with no physical precedent — interactive narratives, navigable databases, motion graphics composites — that could not exist outside the computer

## Key Reference

Alan Kay and Adele Goldberg, "Personal Dynamic Media" (1977): "The computer is a medium that can dynamically simulate the details of any other medium, including media that cannot exist physically."

## Evolution of the Concept

### Implicit use (1995-1999)
Manovich's early arguments about [[Digital Cinema]] and [[Cultural Interface]] implicitly rely on the metamedium concept — the computer transforms cinema and other media because it can simulate and extend them — but the term is not yet foregrounded.

### Explicit theorization (2006)
In [[Alan Kays Universal Media Machine - Manovich 2006]], Manovich provides his fullest treatment, historicizing Kay's concept and arguing that it provides a better theoretical foundation than "new media" or "digital media." The metamedium concept explains why the computer uniquely enables [[Deep Remixability]] and [[Hybrid Media]].

### Software compatibility (2006)
In [[Import Export - Manovich 2006]], Manovich shows how the metamedium works in practice: software compatibility (import/export) between programs allows techniques from different simulated media to combine. The metamedium is realized through the ecosystem of interoperable software tools.
