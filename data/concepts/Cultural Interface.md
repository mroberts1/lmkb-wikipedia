# Cultural Interface

## Definition

Manovich's term for the ways in which computers present and allow us to interact with cultural data — texts, photographs, films, music, virtual environments. Cultural interfaces borrow from three older cultural traditions: cinema, the printed page, and general-purpose HCI (the desktop metaphor).

## Core Thesis

The interface is not a neutral transport mechanism but an active shaper of cultural experience. When cultural content moves to the computer, it is filtered through interface conventions that carry their own ideologies, aesthetics, and assumptions. The cinema screen, the printed page, and the GUI desktop each impose different logics of access, and all three compete within every cultural interface.

## Key Principles

- **Three traditions**: (1) The printed page/book (linear, rectangular, text-oriented); (2) Cinema (rectangular frame, mobile virtual camera, montage); (3) General-purpose HCI (windows, menus, icons, pointers)
- **Cinema dominates**: Manovich argues cinema's conventions increasingly win — the rectangular frame, the virtual camera, montage editing — because cinema already solved problems of presenting dynamic visual information
- **Interface as ideology**: The choice of interface determines what can be said. A database interface encourages browsing; a narrative interface encourages following
- **Screen genealogy**: From Renaissance painting's framed view through cinema's dynamic screen to the computer's interactive screen — each extends and transforms the prior form

## Evolution of the Concept

### Screen archaeology (1995)
In [[Archeology of a Computer Screen - Manovich 1995]], Manovich traces the computer screen's genealogy back through cinema, television, radar, and Renaissance painting — establishing the screen as a persistent cultural form with evolving properties.

### Full formulation (1997)
In [[Cinema as Cultural Interface - Manovich 1997]], the three-tradition framework is presented. Manovich analyzes how websites, CD-ROMs, and virtual worlds negotiate between page, cinema, and HCI conventions.

### Interface genealogy (1997)
In [[Genealogy of the Interface - Manovich 1997]], the concept is expanded with attention to how military interfaces (radar, real-time displays) shaped HCI conventions that later became cultural interfaces.

### Aesthetic interfaces (2006-2007)
In [[Friendly Alien - Manovich 2006]] and [[Information as an Aesthetic Event - Manovich 2007]], Manovich tracks how interfaces evolved from "invisible" (minimizing friction) to explicitly aesthetic experiences — the interface as event rather than tool.
